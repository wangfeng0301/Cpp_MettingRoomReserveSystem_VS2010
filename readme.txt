版本1.0实现功能：
1.账号登录

2.登录后可预约和取消预约。预约成功后主界面右侧列表会将登录者账号填入对应表格中；
取消预约成功后，将对应表格中登陆者账号清除。此版本还未实现对应登陆者只能取消自己之前预约的信息的功能。

3.未登录就预约会弹出对话框提示。

4.左下角单击按钮会弹出使用说明向导。

版本2.0增加功能：

1.登录成功后隐藏登录界面，显示 登陆成功；并显示退出登录按钮；

2.登录者只能取消自己之前预约的；

3.增加.ini文件，每次启动时都会按照上一次的设置初始化。这样之前预约的信息就不会丢失。
